# Java Practice

This workspace is set up for practicing Java. It follows a structure similar to https://github.com/mrinalutkarsh/java-practice.git.

## Structure
- `src/` - Source code
- `test/` - Test code
- `.vscode/` - VS Code settings
- `.github/` - Copilot and GitHub settings

## Getting Started
- Place your Java files in the `src/` directory.
- Add tests in the `test/` directory.
- Customize as needed for your practice.
